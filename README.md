# ByteFlow Pro

Sistema de **inventário e vendas** para loja de eletrônicos. Dashboard com filtros de período, gestão de estoque com reposição automática e carteira de clientes vinculada às vendas. Front-end puro (React 19 + Vite + Tailwind v4) — sem backend, com estado em memória.

## Rodar localmente

**Pré-requisitos:** Node.js

1. Instalar dependências:
   `npm install`
2. Rodar a aplicação:
   `npm run dev`
3. Abrir `http://localhost:3000`

## Scripts

- `npm run dev` — servidor de desenvolvimento
- `npm run build` — build de produção
- `npm run preview` — pré-visualizar o build
- `npm run lint` — checagem de tipos (`tsc --noEmit`)
